About the RSA Theory Test

The real RSA theory test contains 40 questions. You have 45 minutes to complete the test. DTT Ireland's Exam Mode mirrors this exactly.
You need to answer at least 35 out of 40 questions correctly to pass — that is 87.5%. DTT Ireland's mock exam uses the same pass threshold and displays a clear Pass or Fail verdict at the end.
The test is 45 minutes long. DTT Ireland's Exam Mode includes a real countdown timer and a 5-minute warning notification, so you get comfortable with the time pressure before your real test.
The RSA theory test covers 8 categories: (1) Road Signs, (2) Rules of the Road, (3) Traffic Lights, (4) Speed Limits, (5) Hazard Awareness, (6) Vehicle Mechanics, (7) First Aid & Safety, (8) Environment & Economy. DTT Ireland has a dedicated practice mode for every category.
If you fail, you can book another test after a 5-day waiting period. There is no limit on the number of attempts. DTT Ireland's Mistakes Review feature shows you exactly which questions you got wrong after every practice session, helping you target your weak areas so you improve with each attempt.
